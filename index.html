 This project currently in active development.

Development

Build And Install hazelcast-client From Sources

  1. clone repo https://github.com/hazelcast/hazelcast-nodejs-client.git

  2. install global dependencies npm install -g typescript typings

  3. install dependencies npm install

  4. compile TypeScript gulp compile

  5. link package locally npm link

Use Locally Installed Package

  1. cd to project directory

  2. use a locally linked module npm link hazelcast-nodejs-client

  3. follow Usage example

Test

Test prerequisites

  • Java 6+

  • Hazelcast Remote Controller

Usage

JavaScript Example
var HazelcastClient = require('hazelcast-client').Client;
var person = {
    firstName: "Joe",
    lastName: "Doe",
    age: 42
};
var map;
HazelcastClient.newHazelcastClient().then(function (hazelcastClient) {
    map = hazelcastClient.getMap("personMap");
    map.put(1, person).then(function (val) {
        // prints previous value for key `1`
        console.log(val);
    });
    map.get(1).then(function (value) {
        console.log(value);
    })
});
